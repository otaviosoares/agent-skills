# agent-skills

A collection of Claude Code agent skills, developed here and installed via `npx skills`.

## Agent skills

### Issue tracker

Issues live in GitHub Issues on `otaviosoares/agent-skills` (gh CLI); external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical triage labels are used verbatim (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
